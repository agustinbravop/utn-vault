El **Análisis de Datos Empresariales** es el proceso de exploración, transformación y examinación de datos para identificar tendencias y patrones para respaldar el [[Procesos de Toma de Decisiones]].

En empresas, permite obtener una visión más profunda de sus clientes, lo que permite mejorar su experiencia y fomentar su fidelidad a la marca. El ADE es fundamental para tomar decisiones estratégicas y tener:

- Información accionable.
- Identificación de oportunidades y amenazas.
- Optimización de procesos.
- Ventaja competitiva.

Disciplinas actualees:

- Data scientist (usa IA para predicción).
- Data engineer (relacionado a la logística técnica de los datos).
- Data analyst (más relacionado al negocio).
