En [[Ciencia de Datos]], data analytics es un término cargado que se puede dividir en 4 tipos:

1. Descriptivo.
2. Diagnóstico.
3. Predictivo (data science).
4. Prescriptivo (data science).

No se puede hacer buen data science sin buen data analysis.

Habilidades fundamentales para data analysis:

1. Functional skills.
2. Technical skills.
3. Soft skills.
4. Domain knowledge (suele venir de los SMEs, subject matter experts).

Un buen análisis de datos es sistemático y sigue un proceso como [[Gestión de Proyectos de Ciencia de Datos|CRISP-DM]].
