El proceso de [[ETL]] es la parte que más esfuerzo demanda. La ingesta de datos puede ser:

- Batch vs streaming.
- Push vs pull.

Es importante la [[Gestión de Datos Maestros]] y data quality. *DataOps* es un movimiento que busca mejorar la entrega y operación de un [[Producto de Datos]]. Unifica [[Agilidad]], [[5to Nivel/DevOps/DevOps|DevOps]] y el control estadístico de procesos.

Estos conceptos se consideran en la [[Arquitectura de Datos]].
