Conceptos similares a los de [[Estrategia Competitiva]].

$$\text{Análisis estratégico} \longrightarrow \text{Plan estratégico} \longrightarrow \text{Situación deseada}$$

La estrategia empresarial es el modo mediante el cual la organización se ajusta a su entorno para asegurar su supervivencia y crecimiento. Se compone de:

- Objetivos.
- Planes de acción.
- Programas funcionales.
- Recursos requeridos.

Tipos de estrategias según su jerarquía:

1. Estrategia corporativa.
2. Estrategia de negocios (a nivel de unidad de negocios).
3. Estrategia funcional.

No hay acción empresaria sin actores $\implies$ se necesita [[Liderazgo Gerencial]].

## Matriz CAME

La **matriz CAME** define acciones a partir de un análisis FODA para:

- Corregir debilidades.
- Afrontar amenazas.
- Mantener fortalezas.
- Explotar oportunidades.

4 estrategias de la matriz CAME:

1. **Estrategia ofensiva**: unir fortalezas y oportunidades.
2. **Estrategia defensiva**: usar fortalezas para contrarrestar amenazas.
3. **Estrategia de reorientación**: explotar oportunidades si no se puede corregir debilidades.
4. **Estrategia de supervivencia**: corregir debilidades para afrontar amenazas.

Riesgos de no planificar estratégicamente:

- Falta de prioridades.
- Mal uso de recursos.
- Poca adaptación a cambios.

## Planificación Estratégica

Es un proceso sistemático para definir objetivos a largo plazo, analizando el entorno y formulando [[Estrategia de Negocio|estrategias]] para alcanzar los objetivos. Está fundada en la misión, visión y objetivos. Fases:

1. Análisis.
2. Formulación.
3. Implementación.
4. Evaluación y control.

Distintas [[Gestión Gerencial|herramientas gerenciales]] nosayudan en distintas fases.
